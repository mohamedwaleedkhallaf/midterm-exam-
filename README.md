# Beam Analysis and Design Python

## 📌 Project Description
This project performs structural analysis and design checks for beams using Python.

It reads input data from a CSV file and calculates:
- Maximum Bending Moment  
- Maximum Deflection  
- Bending Stress  
- Safety Check (SAFE / NOT SAFE)  

The results are exported to an Excel file.

---

## ⚙️ Features
- Reads beam data from CSV
- Supports multiple beams
- Automatically adapts column names
- Calculates:
  - Bending Moment (kN·m)
  - Deflection (mm)
  - Stress (MPa)
  - Utilization ratio
- Design safety check
- Exports results to Excel
